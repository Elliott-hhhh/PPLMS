# 对比实验设计逻辑思维导图

```
对比实验设计
├─ 实验目标
│  └─ 验证改进PPLM模型在中文对话生成中的角色特征保持能力
├─ 对比框架
│  ├─ 纯模型生成路径
│  │  └─ 直接调用Qwen3-4B的generate方法，无干预生成
│  └─ 改进PPLM干预路径
│     └─ 通过词袋知识注入引导角色特征生成
├─ 数据处理流程
│  ├─ CPED数据集
│  │  └─ 中文电视剧对话数据集
│  ├─ CharacterWordbagManager类
│  │  ├─ 自动扫描对话记录
│  │  ├─ 构建说话人身份映射表
│  │  └─ 分割对话轮次确保上下文完整性
│  ├─ 目标说话者选择
│  │  └─ get_available_speakers方法获取可用说话人列表
│  └─ 数据划分
│     └─ 提取约20,000句对话，按8:2划分为训练/测试集
├─ 角色词袋构建
│  ├─ 自动化提取方法
│  │  ├─ jieba分词器精确分词
│  │  └─ 基于词频统计的特征提取
│  ├─ 多重筛选机制
│  │  ├─ 去除短词（长度<2）
│  │  ├─ 过滤常见停用词
│  │  └─ 设置最小词频阈值（默认1次）
│  └─ 词袋生成
│     └─ 按词频排序，选取前20个高频词作为角色特征词袋
├─ 模型实现
│  ├─ 纯模型生成
│  │  ├─ 调用Qwen3-4B的generate方法
│  │  └─ 参数设置：temperature=0.7，top_k=10，top_p=1.0
│  └─ 改进PPLM模块
│     ├─ 通过subprocess调用generate_ppls.py脚本
│     ├─ 角色词袋作为外部知识注入
│     └─ 多编码支持：优先UTF-8，失败自动切换GBK
├─ 评估体系
│  ├─ 词袋命中率
│  │  └─ 统计生成文本中包含的角色词袋词汇比例
│  ├─ 情感匹配度
│  │  └─ 基于预训练情感分析模型计算情感极性匹配分数
│  └─ 辅助指标
│     ├─ 文本长度
│     └─ 词汇多样性
└─ 实验执行流程
   ├─ run_comparison_experiment函数
   │  ├─ 获取角色词袋
   │  ├─ 对每个提示词运行双路径生成
   │  └─ 固定生成长度（40个token）
   ├─ evaluate_generated_text函数
   │  └─ 对生成结果进行多维度评估
   └─ generate_comparison_report函数
      └─ 计算平均指标并生成对比报告
```

## 可视化说明

1. **实验目标**：明确实验的核心目的是验证改进PPLM模型的角色特征保持能力

2. **对比框架**：采用双路径平行设计，直观对比纯模型与改进PPLM的生成效果

3. **数据处理流程**：从CPED数据集出发，通过CharacterWordbagManager类完成说话人识别、对话分割和数据划分

4. **角色词袋构建**：自动化提取角色特征，通过多重筛选确保词袋质量

5. **模型实现**：详细展示两种生成路径的技术实现细节和参数设置

6. **评估体系**：多维度评估指标，全面衡量生成内容的质量和角色一致性

7. **实验执行流程**：清晰的函数调用关系，确保实验的可重复性和可解释性
